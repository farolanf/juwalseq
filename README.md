# Juwal

Personal project

## Requirements

- Node >= 10.0.0 for full ES 2018 support
- ElasticSearch 6.5+

## Setup

### ElasticSearch
 - install and run ElasticSearch

### Backend

- copy `.env.example` to `.env.development` and configure
- `npm install`
- `npm run reset:dev` to initialize and populate database
- `npm start`

### Frontend

- copy `.env.example` to `.env.development` and configure
- `npm install`
- `npm start`
